# Credit Card Fraud Detection

## Overview

A Machine Learning web application that predicts whether a credit card transaction is fraudulent.

## Features

- Data preprocessing
- One-Hot Encoding
- Feature Scaling
- SMOTE
- Random Forest Classifier
- Streamlit Web App

## Dataset

- 10,000 transactions
- Binary Classification

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## Installation

pip install -r requirements.txt

streamlit run website.py

## Model Performance

Accuracy:  94%
Precision: 86%
Recall:    80%
F1-score:  83%
ROC-AUC:   89.9%
